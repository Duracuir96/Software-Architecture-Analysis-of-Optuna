##  About This Project

This repository documents the architectural analysis of
[Optuna](https://github.com/optuna/optuna),
a state-of-the-art hyperparameter optimization framework
developed by Preferred Networks and published at KDD 2019.

The central research question :

> *"How does Optuna's architecture achieve maximum extensibility
> and scalability while keeping its core API simple enough
> for any ML practitioner to use ?"*
